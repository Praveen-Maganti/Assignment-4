BASIC FORMATTING USING INLINE CSS

Inline CSS is a way to apply styles directly to an HTML element using the style attribute. Instead of writing styles separately in a stylesheet, inline CSS allows you to add CSS rules right inside the tag. This method is useful for quick, specific changes without affecting other elements on the page. With inline CSS, you can control various properties such as text color, font size, background color, and more, all within a single HTML tag. For example, if you want to make a paragraph’s text red or bold without creating a separate class or ID, inline styles let you do this easily and immediately. Although inline CSS is convenient for small, individual adjustments, it is not recommended for styling large websites because it can become hard to maintain and update. Using external or internal stylesheets is generally better for consistency, reusability, and cleaner code. Inline CSS is best suited for quick fixes or testing styles on the fly.
